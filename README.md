
# LessEnergy – Landing Page

LessEnergy es una solución digital enfocada en ayudar a las personas a optimizar y reducir su consumo energético a través de monitoreo, análisis inteligente y recomendaciones personalizadas mediante IA.

## Objetivo del Proyecto
Crear una landing page funcional, clara y atractiva que presente los beneficios que brinda LessEnergy, su propuesta de valor y las herramientas que ofrece para lograr un consumo energético más eficiente.

Este proyecto se desarrolla aplicando GitFlow, buenas prácticas de organización y control de versiones, además de una estructura de carpetas definida para un flujo de trabajo profesional.

## Segmento Objetivo
El producto está dirigido a:

- Personas que sufren por la alta facturación de cobro en su recibo de servicio de electricidad.

- Consumidores vulnerables afectados por cortes de electricidad.

## Principales Características

### ✔ Monitoreo del consumo energético  
Visualización clara del consumo mensual/semanal, con datos fáciles de interpretar.

### ✔ Recomendaciones personalizadas mediante IA  
El sistema analiza hábitos y patrones para sugerir mejoras y posibles anomalías.

### ✔ Alertas y notificaciones  
Avisos sobre consumos inusuales o facturas superiores al promedio.

### ✔ Interfaz sencilla e intuitiva  
Diseñada para usuarios de todas las edades y niveles tecnológicos.

### ✔ Optimización de costos  
Ayuda directa para reducir gastos innecesarios en el uso de energía.
